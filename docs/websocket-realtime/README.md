# WebSocket & Real-Time Communication

## Overview

This folder contains the documentation and implementation proof for the WebSocket & Real-Time Communication module developed for the Chat Platform project.

The implementation was built using:

- Django Channels
- WebSockets
- Redis
- Daphne ASGI Server

---

## Features Implemented

- Real-time messaging using WebSockets
- Dynamic chat room support
- User join notifications
- User leave notifications
- Multiple clients connected simultaneously
- Redis channel layer integration
- Daphne ASGI server configuration

---

## Technologies Used

- Python
- Django
- Django Channels
- Redis
- Daphne
- HTML
- JavaScript
